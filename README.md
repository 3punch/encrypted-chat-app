# Encrypted Chat Application

A simple encrypted chat application built with Python using Socket Programming and AES encryption.

## Features

* Real-time messaging
* AES encrypted communication
* Multi-threading
* Save chat history
* User profile management

## Technologies

* Python
* Socket Programming
* Threading
* PyCryptodome

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python chat.py
```
